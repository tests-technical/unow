# Parte 2: Creación de una plantilla base de WordPress

1. Crear una plantilla base personalizada en WordPress que incluya:

    - Encabezado (header) con menú principal y logo.

    - Pie de página (footer) con información de copyright y contacto.

    - Estilos básicos de CSS para una apariencia limpia y consistente.

2. Asegurarse de que la plantilla base sea responsive y se adapte correctamente a diferentes dispositivos.

# Parte 3: Maquetación de la landing page con Elementor

1. Utilizar Elementor para crear la landing page siguiendo el diseño proporcionado en el archivo Figma: [Diseño](https://www.figma.com/design/BT8HX2ooNoITmjtz139o0r/Prueba-de-Maquetaci%C3%B3n)

2. Implementar todos los elementos del diseño, incluyendo:

    - Imágenes y textos.
    - Botones.
    - Secciones con diferentes estilos y animaciones.

3. Asegurarse de que la maquetación sea fiel al diseño original y se vea bien en diferentes dispositivos.

# Parte 4: Optimización y publicación de la landing page

1. Optimizar la landing page para SEO (Search Engine Optimization), incluyendo:

    - Títulos y meta descripciones adecuados para cada sección.
    - Imágenes con alt text descriptivo.
    - URL amigables (slugs) para las páginas.

2. Publicar la landing page en el sitio web de WordPress.

# Parte 5: Creación de un plugin de WordPress

1. Desarrollar un plugin de WordPress que agregue un shortcode para mostrar un formulario de contacto en cualquier página o entrada.

2. El formulario debe incluir campos para nombre, correo electrónico y mensaje.

3. El plugin debe guardar los datos del formulario en la base de datos de WordPress.

4. Los datos almacenados en la base de datos de WordPress que provienen del formulario deben poder ser visibles en una página del administrador de WordPress en forma de listado.

5. El plugin debe estar bien documentado con comentarios de código y un archivo README.

6. El código del plugin debe seguir las mejores prácticas de desarrollo de plugins de WordPress.

7. El código del plugin debe ser limpio, eficiente y fácil de entender.

# Entrega

1. Enviar el paquete de instalación de Duplicator (.zip) de la landing page como archivo adjunto.

2. Incluir un archivo README.txt con instrucciones para instalar la landing page.

3. Enviar el plugin de WordPress como un archivo ZIP independiente.

4. Incluir un archivo README.txt con instrucciones para instalar y usar el plugin.
